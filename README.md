# Invitation Maker v2

A dynamic invitation generator built with **Flask (Python backend)** and **JavaScript + CSS (frontend)**.

## Features
- User signup and login
- Choose from 5 event categories (Wedding, Birthday, Anniversary, Party, Corporate)
- Multiple templates per event (like Canva)
- Edit invitation details with live preview
- Export invitations as PDF, JPG, PNG, or DOCX
- Send invitations via email
